Job Application Factory

A local-first AI career copilot that drafts job-application assets from approved evidence, then checks for unsupported claims before export.

Problem

Generic AI resume tools create shallow, risky application materials. The useful version needs to stay grounded in real accomplishments, preserve privacy, and adapt the story to each role.

Solution

Job Application Factory uses a private evidence library, local/demo LLM workflows, semantic job matching, package generation, and critic checks. The public demo runs in Mock Demo Mode using a fictional profile.

What It Proves

  • RAG-grounded generation instead of unsupported claim invention.
  • Multi-agent workflow design with critic/self-correction behavior.
  • Streamlit product design across onboarding, discovery, packages, and tracking.
  • Privacy-aware local data boundaries through a configurable data directory.

Verification

Current audit: key dependencies imported successfully. The app launched with a temporary `JAF_DATA_DIR`, Mock Demo Mode was enabled, Alex Builder demo profile was visible, and Live Mode was not active in the screenshot session.

Redaction Note
