Problem

AI-assisted development creates many ideas quickly. Without a durable evidence trail, it is easy to chase the newest thought instead of testing the highest-value one.

Solution

The engine records each idea as a structured lifecycle: idea, hypothesis, test plan, test run, result, decision memo, asset, and promoted project. It stays local-first with SQLite and a lightweight Python backend.
